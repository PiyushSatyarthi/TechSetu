TechSetu
TechSetu is an agri-tech platform designed to help Indian farmers list crops, manage produce, and connect with buyers through a simple, efficient, and farmer-friendly digital experience.
It combines a farmer dashboard, crop listing workflow, authentication, and realtime features into one platform built for practical use in the field.

Features
Farmer dashboard for viewing and managing listings.
Add-listing workflow for submitting crop details.
Crop photo upload support.
Firebase Authentication for secure login and signup.
Realtime data updates for live dashboard changes.
Map-based tracking support for live location visualization.
Responsive interface for desktop and mobile.
Clean agricultural UI with green and gold branding.

Project Goals
TechSetu aims to:
Make crop listing easier for farmers.
Reduce dependency on middlemen.
Improve visibility of crop availability.
Support transparent pricing and buyer discovery.
Provide a simple and modern interface that works well on mobile devices.
